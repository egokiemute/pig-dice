What is Pondice?

Pondice is a simple and engaging dice game developed as a final project for the course Web Technologies under the guidance of Prof. Boyka Gradinarova and Assist. Madlen Stoycheva. The game was built using HTML, CSS, and JavaScript, showcasing core web development concepts such as user interaction, game logic, responsive design, and dynamic content manipulation.

More than just a course requirement, Pondice was inspired by family. My grandmother once mentioned that she wanted a fun way to improve her confidence with numbers while becoming more comfortable with technology. Designed so she and my niece could enjoy playing together on an iPad — learning and entertainment going hand in hand.
